# Анализ древесно-торфяного композита

Приложение для анализа изображений древесно-торфяного композита с графическим интерфейсом на Python.

## Описание

Приложение позволяет анализировать изображения древесно-торфяного композита и определять процентное содержание торфа. На основе анализа предоставляются рекомендации по применению материала в зависимости от его характеристик (теплоизоляция, звукопоглощение, прочность).

## Возможности

- **Анализ цельного образца композита** - определение процентного содержания торфа в образце
- **Анализ композита в разрезе** - анализ структуры материала в поперечном сечении
- **Автоматическая валидация** - проверка на наличие людей на изображении и соответствие изображения древесно-торфяному композиту
- **Визуализация результатов** - отображение обработанного изображения с выделенными контурами
- **Экспорт отчетов** - сохранение результатов анализа в формате Word (.docx)

## Технологии

- **Tkinter** - графический интерфейс пользователя
- **OpenCV** - обработка и анализ изображений
- **PIL/Pillow** - работа с изображениями
- **python-docx** - создание Word документов
- **NumPy** - численные вычисления

## Особенности анализа

Приложение определяет процентное содержание торфа и классифицирует материал по трем категориям:

- **10-40% торфа**: Идеален для теплоизоляции
- **40-60% торфа**: Баланс между теплоизоляцией, звукопоглощением и прочностью
- **60%+ торфа**: Оптимален для звукоизоляции и прочности

## Примечания

- Приложение работает только с PNG изображениями
- На изображении не должно быть людей (автоматическая проверка)
- Изображение должно содержать древесно-торфяной композит (автоматическая валидация)
